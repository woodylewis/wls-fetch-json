This app uses a controller and service to make RESTful calls to my blog (running on a remote Drupal CMS), and retrieves either a list of all published posts or a post from that list. Angular promises are used to handle return values, and strict contextual escaping is used on individual post content.


The parent controller stores the list in its scope, and a child controller processes the specific post. The blog domain is prepended to URLs for all images hosted on that domain. Some older articles have different styles.
